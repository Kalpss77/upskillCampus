# Banking Information System

## Project Overview
The Banking Information System is a Java-based console application developed as part of the USC–UCT Internship program. The project demonstrates core Java concepts such as classes, objects, methods, conditional statements, loops, and user input handling.

## Features
- Create bank account
- Deposit amount
- Withdraw amount
- Check account balance
- Simple menu-driven interface

## Technologies Used
- Java
- GitHub

## How to Run
1. Clone the repository
2. Open the project in any Java-supported IDE
3. Compile and run the `BankingInformationSystem.java` file

## Internship Details
- Internship Platform: USC / UCT
- Project Type: Java Console Application
